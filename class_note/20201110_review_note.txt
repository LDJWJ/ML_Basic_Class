05 주피터 노트북 실행키
   CTRL + ENTER : 셀을 실행하고 그대로 현재 셀에 머문다.
   SHIFT + ENTER : 셀을 실행하고 아래에 하나의 셀을 추가
   ESC + M : 설명을 다는 셀로 동작하도록 한다.
   ESC + Y : 코딩이 동작하는 셀로 동작하도록 한다.
   ESC + A : 현재 위치하고 있는 셀의 위에 하나의 셀이 추가 
   ESC + B : 현재 위치하고 있는 셀의 아래에 하나의 셀이 추가 
   ESC + X : 현재 위치하고 있는 셀을 삭제
