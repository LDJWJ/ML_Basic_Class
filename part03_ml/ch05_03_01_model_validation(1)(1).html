분류의 평가지표를 살펴보자.¶

정확도(accuracy) : 정확하게 예측/전체 예측수¶

\begin{equation} \text{accuracy} = \frac{\text{TP} + \text{TN}}{\text{TP} + \text{FP} + \text{FN} + \text{TN}} \end{equation}

정밀도(precision) : 예측을 양성(Positive)으로 한것 전체-TP+FP중에 맞는 것(T*)¶

\begin{equation} \text{(precision)} = \frac{\text{ (TP)}}{ \text{ (TP+FP) } } \end{equation}
  • ? : (FP)
    • (positive),

민감도(sensitivity), 재현율(recall, TPRate)¶

  • (TP + FN) ?(TP)
  • TP/(TP + FN)
\begin{equation} \text{(recall, )} = \frac{\text{ (TP)}}{ \text{ (TP+FN) } } \end{equation}
  • FP .
  • ? FN( . .) , .
  • ( (-Negative), . ** (Positive),
    • (Negative)** ?
  • (sensitivity), (hit rate), (TPR) .

특이도¶

  • (FP + TN) ?(TN)
  • TN/(FP + TN)
\begin{equation} \text{} = \frac{\text{ (TN)}}{ \text{ (FP + TN) } } \end{equation}

FPRate¶

  • (FP + TN) ?(FP)
  • FP/(FP + TN)
\begin{equation} \text{FPRate} = \frac{\text{ (FP)}}{ \text{ (FP + TN) } } \end{equation}
